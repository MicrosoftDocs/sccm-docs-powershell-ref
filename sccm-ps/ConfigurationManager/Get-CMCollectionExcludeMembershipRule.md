# Get-CMCollectionExcludeMembershipRule

## SYNOPSIS
Gets a collection exclude membership rule

## SYNTAX

### ByNameAndName (Default)
```
Get-CMCollectionExcludeMembershipRule -CollectionName <String> [-ExcludeCollectionName <String>]
 [<CommonParameters>]
```

```

 [<CommonParameters>]
```
