# Get-CMTSStepInstallApplication

## SYNOPSIS
Gets a t s step install application.

## SYNTAX

### ByValue (Default)
```
Get-CMTSStepInstallApplication [-InputObject] <IResultObject> [-StepName <String>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

